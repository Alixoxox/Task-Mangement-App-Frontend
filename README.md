## üöÄ Getting Started

### Prerequisites
* [Node.js](https://nodejs.org/) (v18 or higher recommended)
* An active backend (Link to your [Backend Repo](https://github.com/Alixoxox/Task-Management-App-Backend))

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Alixoxox/Task-Management-App-Frontend.git
    cd Task-Management-App-Frontend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Setup Environment Variables:**
    Create a `.env` file in the root directory and add your backend URL:
    ```env
    VITE_BACKEND_URL=https://task-management-app-backend-k7o1.onrender.com
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```
